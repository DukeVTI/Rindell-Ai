CAUSE 1: Disk Full
------------------
Symptom: npm says success but files don't appear
Check: df -h .
Fix: Free up space, then npm install

CAUSE 2: Permission Issues
---------------------------
Symptom: "EACCES" errors or modules missing
Check: ls -la | grep node_modules
Fix: sudo chown -R $USER:$USER ~/rindell

CAUSE 3: Wrong npm Configuration
---------------------------------
Symptom: npm installs to ~/.npm instead of ./node_modules
Check: npm config list | grep prefix
Fix: npm config delete prefix && npm install

CAUSE 4: Corrupted npm Cache
-----------------------------
Symptom: Installation completes but modules not usable
Check: npm cache verify
Fix: rm -rf ~/.npm && npm cache clean --force

CAUSE 5: Multiple Node Installations
-------------------------------------
Symptom: npm and node from different sources
Check: which node && which npm && node -v && npm -v
Fix: Use NVM to manage Node.js (see Option D above)
